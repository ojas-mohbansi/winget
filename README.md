# Winget Management App

A modern, fast, and weightless Web UI for managing Windows Package Manager (winget).

## Features
- Search and install packages
- Manage installed applications
- Modern and responsive design
- Integrated with Tauri for a lightweight desktop experience

## Getting Started
To get started with development, follow these steps:
1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run tauri dev`

## Release
The project is automatically packaged and released via GitHub Actions.
Check the [Releases](https://github.com/ojas-mohbansi/winget/releases) page for the latest version.
